# Diseño de esquema de compresión Huffman

Contexto: Necesito comprimir un archivo de texto con las siguientes características:

-   Idioma: español
-   Tipo: [especificar: código fuente / texto literario / datos tabulares / logs]
-   Tamaño aproximado: [especificar]

Tarea:

1. Estima las frecuencias típicas de caracteres para ese tipo de texto en español.
2. Construye un árbol de Huffman basado en esas frecuencias estimadas.
3. Genera la tabla de códigos resultante.
4. Calcula la compresión teórica esperada vs ASCII.
5. Explica cómo se guardaría el árbol en el archivo comprimido.

Pregunta adicional: ¿Qué pasaría si el texto real tiene frecuencias muy diferentes a las estimadas?
