# Comparación de estructuras para indexación

Contexto: Estoy diseñando un sistema y debo elegir la estructura de índice adecuada.

Escenario:

-   Tabla con [especificar cantidad] de registros
-   Operaciones frecuentes: [búsquedas puntuales / rangos / inserciones frecuentes]
-   Almacenamiento: [SSD / HDD / memoria RAM]
-   Tamaño de bloque de disco: 4KB

Tarea:

1. Compara BST, AVL, B-tree y B+ tree para este escenario.
2. Calcula para cada uno:
    - Altura esperada
    - Accesos a disco por búsqueda
    - Ventajas/desventajas específicas
3. Recomienda la estructura óptima con justificación.

IMPORTANTE: Muestra los cálculos para que yo pueda verificarlos.
